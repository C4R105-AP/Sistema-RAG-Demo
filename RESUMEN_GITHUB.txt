------------------------------------

3. Público ✅
4. NO marcar "Add README"
5. Click "Create repository"

PASO 2: Subir el código (PowerShell)
------------------------------------

git init

git commit -m "Initial commit: Sistema RAG con FAISS y LLMs"
git remote add origin https://github.com/TU_USUARIO/Sistema-RAG-Demo.git
git branch -M main
git push -u origin main

PASO 3: Crear Release con el ejecutable
----------------------------------------
1. En GitHub, ve a tu repo
2. Click "Releases" (lado derecho)
3. Click "Create a new release"
4. Tag: v1.0.0
5. Title: "Sistema RAG v1.0.0 - Windows"
6. Arrastra: Sistema_RAG_Portable.zip
7. Click "Publish release"
